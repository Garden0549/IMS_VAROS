## *『关于模块』*

- ### 模块功能
> 有时你可能发现使用第三方调度（主要分为两类：CPU调频调度 CPU线程放置）功耗更高，帧率更不稳定，甚至无法调节CPU频率，原因之一是：官方的调度在干扰第三方调度
> 
> **而本模块用于屏蔽ColorOS官方调度，阻止其干扰第三方调度**

- ### 适用范围
> 系统：ColorOS12～16或RelameUI3.0～7.0

- ### 注意事项
> ①安装模块前，一定要先安装救砖模块
> 
> ②卸载模块即可恢复模块所做所有改动（除了清空的应用增强服务数据）
  
- ## *再次提醒*
> ***安装救砖模块，以避免无法开机***
